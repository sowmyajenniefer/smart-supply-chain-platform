# Smart Supply Chain Order Management Platform

An enterprise-style full-stack portfolio project built to demonstrate backend architecture, supply chain workflows, event-driven design, and cloud-native deployment.

## Project Goal

This platform manages product catalog, inventory availability, order processing, payment simulation, and notifications for a supply chain/order management system.
